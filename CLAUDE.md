# Project Documentation

## AEGIS Integration

This project uses AEGIS as a submodule. Available commands:

### Executive Commands
- `/ceo` - AI_CEO (Strategic decisions)
- `/cto` - AI_CTO (Technical architecture)
- `/cfo` - AI_CFO (Financial analysis)
- `/cro` - AI_CRO (Risk assessment)
- `/cmo` - AI_CMO (Marketing strategy)
- `/cpo` - AI_CPO (Product management)
- `/cdo` - AI_CDO (Design leadership)
- `/investor` - AI_Investor (Investment analysis)

### Specialist Commands
- `/impl` - Implementation agent
- `/qa` - Quality assurance
- `/sec` - Security review
- `/tech` - Technical review
- `/design` - Design tasks
- `/ux` - UX design
- `/ui` - UI design

### Red Team Commands
- `/advocate` - Devil's Advocate
- `/skeptic` - Skeptic analysis
- `/redteam` - Both Red Team agents

### Tool & Workflow Commands
- `/uipro` - UI/UX Pro Max design intelligence
- `/aegis-os` - Organization pipeline (6 orgs, 24 agents)
- `/multi-llm` - Multi-LLM orchestrator (Claude + Gemini + Codex)
- `/review-llm` - Multi-LLM cross-review
- `/codex` - Delegate to OpenAI Codex
- `/gemini` - Delegate to Google Gemini
- `/tdd` - Test-Driven Development workflow
- `/build-fix` - Build error resolution
- `/refactor-clean` - Refactor & cleanup

For full AEGIS documentation, see: `aegis/CLAUDE.md`
